# Reaper-automatic-random-generation
This is a script in LUA for Reaper to apply audio effects to an audio clip with multiple random parameters.

It applies the following effects:

|     Effect        |     Params            |     Range            |
|-------------------|-----------------------|----------------------|
|     Chorus        |     Length            |     0 – 200          |
|                   |     Voices            |     0 – 8            |
|                   |     Wet Mix           |     30 – 42          |
|                   |     Dry Mix           |     30 – 42          |
|                   |     Rate              |     0 – 16           |
|                   |     Pitch             |     0 – 1            |
|     Distortion    |     Gain              |     10 – 40          |
|                   |     Hardness          |     1 – 10           |
|     EQ            |     Gain 1            |     0 – 0,75         |
|                   |     Gain 2            |     0 – 0,75         |
|                   |     Gain 3            |     0 – 0,75         |
|                   |     Bandwidth 1       |     0 – 1            |
|                   |     Bandwidth 2       |     0 – 1            |
|                   |     Bandwidth 3       |     0 – 1            |
|     Rever         |     Wet Mix           |     0 – 3            |
|                   |     Dry Mix           |     0 – 3            |
|                   |     Room              |     0 – 1            |
|                   |     Dampening         |     0 – 1            |
|                   |     Lowpass           |     0 – 1            |
|                   |     Highpass          |     Paso bajo – 1    |
|     Flanger       |     Length            |     0 – 200          |
|                   |     Feedback          |     -120 – 6         |
|                   |     Wet Mix           |     -30 – 12         |
|                   |     Dry Mix           |     -30 – 12         |
|                   |     Rate              |     0 – 100          |
